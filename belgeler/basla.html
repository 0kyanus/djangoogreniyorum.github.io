İlişkili-Nesne İmgeleyici (Object-relational mapper)

Veri kalıplarınızı baştan sona Python'da çözün. Ücretsiz olarak zengin, dinamik bir veritabanı erişimi API'si elde edersiniz, ancak gerekir ise SQL yazabilirsiniz.

Devamını Oku class Band(models.Model): """A model of a rock band.""" name = models.CharField(max_length=200) can_rock = models.BooleanField(default=True) class Member(models.Model): """A model of a rock band member.""" name = models.CharField("Member's name", max_length=200) instrument = models.CharField(choices=( ('g', "Guitar"), ('b', "Bass"), ('d', "Drums"), ), max_length=1 ) band = models.ForeignKey("Band")
